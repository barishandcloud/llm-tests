# LLM Tests Repository

This repository contains various tests for large language models.

## Contents
- Basic prompts and interactions
- Benchmarking examples
- Validation cases
- Configuration files

## Getting Started
1. Clone the repository: `git clone https://github.com/barishandcloud/llm-tests.git`
2. Explore the test cases in the `tests/` directory
3. Modify and run tests according to your needs
